# SanoTips

Live sports prediction platform using Firebase Firestore.

## Features
- View free predictions
- Admin panel to add matches
- Firestore & Firebase Authentication
- VIP subscription support (safe/fixed)

## Hosting
1. Install Firebase CLI: `npm install -g firebase-tools`
2. Login: `firebase login`
3. Initialize: `firebase init hosting`
4. Deploy: `firebase deploy`

## Firebase Config
Replace config in `public/js/firebase.js` with your own Firebase project.
